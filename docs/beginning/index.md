# 欢迎来到东华高中毕业生大学指北 🎓

## 📚 文档作用

这是一个由东华高中毕业生共同维护的**大学信息分享平台**。我们收集整理了各地知名高校的真实就读体验、专业介绍、校园生活、报考建议等内容，旨在为学弟学妹们的升学选择提供第一手参考资料。

- 如果你想阅读这里的文章，请先查看[阅读者使用方法](/beginning/reading);
- 如果你希望投稿，请先查看[编辑者使用方法](/beginning/writing);
- 如果你希望联系文档的维护者，请看[联系方式](/beginning/contact);

## ⚠️ 声明

**这一部分请务必阅读**

1. **内容仅供参考**：本站所有内容均来自东华高中毕业生的个人经验分享，代表作者个人观点，不代表学校官方立场。
2. **信息时效性**：大学政策、专业设置、录取标准等会随时间变化，请以各高校官方网站的最新信息为准。
3. **个体差异性**：每个人的体验和感受因个人背景、兴趣、能力而异，请结合自身情况理性参考。
4. **非官方平台**：本站为学生自发维护的非盈利性质文档，与各高校无正式关联。禁止任何个人或组织利用本文档盈利。
5. **维护者水平有限**：本站出现的任何 bug 均来自维护者的水平不足，请多多见谅。
