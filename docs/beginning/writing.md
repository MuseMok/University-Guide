# 🖊 编辑者使用方法

我们欢迎所有东华高中的学长学姐参与内容贡献！无论您就读于中国大陆，或者中国港澳台，再或者是国外的大学，无论您的学校层次如何，我们都欢迎您为本站参与内容贡献。

## 内容规范

- **您可提交的内容**：本站希望提供学长学姐们对您所就读的大学的客观条件与主观感受；同时本站还希望还可以提供学长学姐们的一些感悟，这一部分我们会放置到**其他**专栏中，建议以一篇文章的形式展现（可以参考专栏中已经存在的文章）。
- **您贡献的内容务必真实**：如果是与学校客观条件或者主观感受的内容，您攥写的内容请保证**真实性**。同时，应当表明您的年级以保证**时效性**。
- **您贡献的内容请使用结构化组织**：建议使用标题、列表等让内容清晰易读。
- **您贡献的内容不要暴露任何人的隐私**：不要透露个人敏感信息或他人隐私。
- **您贡献的内容不得出现任何违法言论**：请您保持客观理性，避免攻击性言论、辱骂性言论，同时避免涉及任何带有政治的内容。

## 贡献流程

- **编写内容**：请将您需要投稿的内容编辑到 word 文档，或者 markdown 文档中（最好是 markdown 文档）。如果可以，请您在您编写的内容的最开始加上您攥写内容的主题。
- **文档命名**：如果您投稿的是全新的内容，请将您的文档命名为“投稿-您的大学入学年份-您的专业-您的署名”；如果是对已经存在的内容进行补充或者勘误，请将您的文档命名为“补充/勘误-您想要补充或者勘误的内容的所在位置-您的署名”。
- **发送邮件**：请将文件发送到邮箱 University_Guide@163.com ，主题就是文档名字。

为了保证本站的安全，本项目的源码仅公开可见，暂时不支持其他账户直接修改本站的仓库，请多多见谅。

## 审核

- 所有贡献内容需要通过审核才能被上传到文档中。
- 维护者会审核内容的**真实性、规范性和准确性**。
- 一般来说，维护者会将审核通过的内容**原封不动**上传至文档。
- 通常会在**周日**统一进行审核上传，项目前期更新可能会比较频繁。
