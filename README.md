# 东华高中毕业生大学指北 🎓

一个为东华高中毕业生提供大学生活指导和经验分享的文档网站。

## 📖 项目简介

本项目旨在为东华高中的毕业生们提供真实、实用的大学生活指南。通过学长学姐们的经验分享，帮助即将步入大学的同学们更好地了解各所大学的真实情况，做出更明智的选择。

## 🌟 特色内容

- 📚 学校概况与专业介绍
- 🏠 校园生活体验分享
- 💡 学习与发展建议
- 🗂️ 院校索引库 - 按省份快速查找目标院校

## ✍️ 投稿规范

如果你想分享自己的大学经历，请注意：

1. **标签第一位必须是学校名**（如：武汉大学、中山大学）
2. **必须包含省份标签**（如：湖北、广东、山东）
3. 其他标签可自由添加

详细投稿指南请访问：[贡献指南](https://musemok.github.io/University-Guide/docs/writing)
