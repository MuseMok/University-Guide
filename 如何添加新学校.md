# 如何添加新学校文章

现在系统已经实现了**完全自动化**识别，通过文章标签自动识别学校和省份！

## 📝 添加新文章的步骤

### 1. 创建新的 Markdown 文件

在 `blog/` 目录下创建文章，例如：

```bash
blog/2026-02-05-beijing-pku-1.md
```

### 2. 在文章头部添加标签（包含学校名和省份名）

**重要规则**：
- ✅ **必须包含学校名称**（包含"大学"、"学院"或"学校"关键字）
- ✅ **必须包含省份标签**（可以是省份全称如"广东省"，或简称如"广东"、"北京"）

```markdown
---
slug: beijing-pku-1
title: 北京大学生活分享
authors: [your_name]
tags: [北京大学, 计算机, 校园生活, 北京]
---

你的文章内容...
```

### 3. 就这样！

**不需要修改任何其他文件！** 系统会自动：
- ✅ 从标签中识别出"北京大学"（包含"大学"关键字）
- ✅ 从标签中识别出"北京"省份
- ✅ 自动将"北京大学"归类到"北京市"
- ✅ 在"院校索引库"页面中显示
- ✅ 点击该学校卡片会跳转到所有相关文章

## 🏷️ 标签规则

### 学校标签
包含以下任一关键字的标签会被识别为学校：
- **大学**：北京大学、清华大学、复旦大学...
- **学院**：北京电影学院、中央音乐学院...
- **学校**：XX职业学校...

### 省份标签
支持的省份格式（任选其一即可）：
- **全称**：`广东省`、`北京市`、`内蒙古自治区`
- **简称**：`广东`、`北京`、`内蒙古`
- **简写**：`粤`、`京`、`蒙`

### 标签示例

#### ✅ 正确示例
```markdown
# 示例1：使用省份简称
tags: [中山大学, 旅游管理, 校园生活, 广东]

# 示例2：使用省份全称  
tags: [武汉大学, 校园生活, 湖北省]

# 示例3：多个主题标签
tags: [山东大学, 计算机, 保研, 青岛, 山东]

# 示例4：学院也可以
tags: [北京电影学院, 表演系, 北京]
```

#### ❌ 错误示例
```markdown
# 缺少省份标签
tags: [北京大学, 计算机]  # ❌ 没有省份信息

# 学校名称不完整
tags: [北大, 计算机, 北京]  # ❌ "北大"不包含"大学"关键字

# 只有省份没有学校
tags: [校园生活, 北京]  # ❌ 没有学校标签
```

## 🔄 工作原理

1. **读取文章**: 系统自动读取所有博客文章的标签
2. **识别学校**: 查找包含"大学/学院/学校"的标签
3. **识别省份**: 匹配 37 个省级行政区的名称
4. **自动关联**: 将同一篇文章中的学校和省份关联起来
5. **智能分类**: 按省份组织学校，有文章的省份优先显示

## 📊 实际案例

假设你添加了以下文章：

```markdown
<!-- blog/2026-02-06-zhejiang-zju-1.md -->
---
slug: zhejiang-zju-1
title: 浙江大学计算机专业分享
authors: [student]
tags: [浙江大学, 计算机, 求是园, 浙江]
---

我的大学生活...
```

系统会自动：
1. ✅ 识别标签"浙江大学"（包含"大学"）
2. ✅ 识别标签"浙江"（匹配"浙江省"）
3. ✅ 在"院校索引库"的"浙江省"分类下显示"浙江大学"
4. ✅ 点击后跳转到 `/blog/tags/浙江大学` 显示所有相关文章

## ✨ 优势

- ✅ **零配置**: 只需添加文章和标签，无需修改代码
- ✅ **自动识别**: 智能识别学校名称和省份信息
- ✅ **灵活扩展**: 支持任何包含"大学/学院/学校"的机构
- ✅ **多省份支持**: 一个学校可以关联多个省份（如有多个校区）
- ✅ **智能排序**: 有文章的省份自动排在前面
- ✅ **完全动态**: 无需维护学校到省份的映射表

## 🚀 开始写作

现在你可以直接在 `blog/` 目录下创建新文章，添加学校和省份标签，然后提交即可！

```bash
# 1. 创建文章
code blog/2026-02-06-你的学校-1.md

# 2. 添加内容和标签（记得包含学校名和省份名）
# 3. 提交
git add .
git commit -m "添加XX大学文章"
git push
```

## 💡 提示

### 推荐标签顺序
建议按照以下顺序添加标签，便于阅读：
```markdown
tags: [学校名, 专业/主题, 其他关键词, 城市, 省份]
```

### 多校区情况
如果学校有多个校区，可以添加多个省份标签：
```markdown
tags: [中国科学技术大学, 物理, 合肥, 安徽, 苏州, 江苏]
```
系统会自动在两个省份下都显示这所学校。

### 特殊情况
- **直辖市**: 使用"北京"、"上海"、"天津"、"重庆"即可
- **自治区**: 使用简称如"内蒙古"、"新疆"、"西藏"、"宁夏"、"广西"
- **港澳台**: 使用"香港"、"澳门"、"台湾"

就这么简单，专注写作即可！🎉
